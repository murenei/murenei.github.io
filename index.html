Examinyzer

Maths web app for students, launched April 2017

Built using Django, Python, Bootstrap, JS and MathJax.

Generates maths examples for certain problem types that students can use as exam preparation.

The project was originally conceived as an "engineering growth hack" for Tutify (my online tutoring company), which would consume past HSC exam papers in pdf form, extract the questions, answers and marking guidelines using OCR (optical character recognition) to create a large database of practice materials for students. This immediately ran into copyright issues and so I resolved to generate the questions themselves instead (using Python functions).
