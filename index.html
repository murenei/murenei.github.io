Projects

Examinyzer

Maths web app for students, launched April 2017

Built using Django, Python, Bootstrap, JS and MathJax.

Generates maths examples for certain problem types that students can use as exam preparation (hence "Examinyzer").

The project was originally conceived as an "engineering growth hack" for Tutify (my online tutoring company), which would consume past HSC exam papers in pdf form, extract the questions, answers and marking guidelines using OCR (optical character recognition) to create a large database of practice materials for students. This immediately ran into copyright issues and so I resolved to generate the questions themselves instead (using Python functions).

Topic.ly

Machine Learning web app, coming July 2018!

Using Machine Learning to classify articles and their authors, and map a bipartite network graph of authors and their articles. The project allowed me to practice skills in web scraping, machine learning, deep learning, natural language processing, and network analysis.

Uses Django, JS and Bootstrap as well as the following Python packages: Scrapy, NetworkX, scikit-learn, numpy, pandas

Network graphs plotted using Cytoscape. Other visualisations use plot.ly
